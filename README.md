Crowbook-bench
--------------

Benchmarks for Crowbook, used to render "Le chien des Baskerville" in
HTML, EPUB and LaTeX for various versions of Crowbook, to see 
performance evolution.

Usage:

```
$ cargo bench 2> /dev/null
```
